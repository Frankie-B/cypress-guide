# Webapp 

## Setup
* `npm install`

## Dev
* `npm start` 

And visit `http://localhost:8080`

# Contributing 
Guidance on depdendencies: 
* Transpiler:
  * typescript
* Frontend Dev:
  * webpack 
  * webpack-dev-server
  * ts-loader
  * html-webpack-plugin
  * clean-webpack-plugin
* Backend Dev: 
  * ts-node 
* For frontend execution: 
  * react 
  * react-dom 
  * mobx 
  * mobx-react 
  * typestyle
  * csstips
* For backend execution: 
  * express 
  * lowdb
* For both Frontend and Backend execution: 
  * axios
