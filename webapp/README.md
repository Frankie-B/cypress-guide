# Webapp 

## Setup
* `npm install`

## Dev
* `npm start` 

And visit `http://localhost:8080`

# Contributing 
Guidance on depdendencies: 
* Transpiler:
  * typescript
* Frontend Dev:
  * webpack / webpack-dev-server / ts-loader / html-webpack-plugin / clean-webpack-plugin
* Backend Dev: 
  * ts-node 
* For backend execution: 
  * express 
  * lowdb
* For both Frontend and Backend execution: 
  * axios
